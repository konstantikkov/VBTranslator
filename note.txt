Транслятор Visual Basic.
Организация ввода:
console - после данного ключевого слова можно осуществлять ввод кода с новой строки
для окончания ввода пользователю следует ввести ключевое слово EOF
file *относительный путь к файлу* - трансляция кода из файла.

Примеры исполняемого кода в папке examples.
Код программы в папке VBtranslator
